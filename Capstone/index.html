Simply Travel


Simply Travel is a travel-based website that helps people plan the perfect vacation!


This website utilizes four different APIs, which are in relation to travel, to assist the user for organizing a trip.
The APIs used are shown in this order respectively - the Amadeus API to retrieve flight information such as ticket cost,
the OpenWeatherMap API to tell the user what the weather is like in the city they want to travel to, the Google Maps API
to show the location of the vacation spot, the Avis Budget Group API to present the location and contact information of car
rental places in the city that the user picks!


We have created this website prototype to showcase the conjoined information for two cities:
Philadelphia and Miami!
Click on the tabs to check it out!
